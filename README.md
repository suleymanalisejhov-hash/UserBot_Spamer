# Telegram UserBot (Multi-Account)

Юзербот на Pyrogram с поддержкой нескольких аккаунтов.

## Деплой на Railway

1. Подключи этот репозиторий на [railway.app](https://railway.app)
2. В Variables добавь **только**:

| Переменная | Описание |
|---|---|
| `SESSION_STRING` | Сессия первого аккаунта |
| `SESSION_STRING1` | Сессия второго аккаунта (опционально) |
| `SESSION_STRING2` | Сессия третьего аккаунта (опционально) |

> `API_ID` и `API_HASH` уже вшиты в код.

## Как получить SESSION_STRING

Используй [@SessionStringGeneratorBot](https://t.me/SessionStringGeneratorBot) или аналоги (SessionPhazerBot и т.п.).

## Команды

| Команда | Что делает |
|---|---|
| `/spam текст` | Спамит каждую минуту |
| `/spam 5с текст` | Спамит каждые 5 минут (1с = 1 минута) |
| `/stopspam` | Останавливает спам в текущем чате |
| `/setmsg1 текст` | Меняет 1-е сообщение автоответчика |
| `/setmsg2 текст` | Меняет 2-е сообщение автоответчика |
| `/msgs` | Показывает текущие тексты автоответчика |
| `/getmedia @channel` | Скачивает всё медиа из канала/группы целиком |
| `/stopmedia` | Останавливает текущее скачивание |
| `/getmed <ссылка на сообщение>` | Скачивает медиа только из одного сообщения/поста по ссылке (если это альбом — скачивает весь альбом целиком) |

**Slowmode** подстраивается автоматически.

**Уведомления в Избранное:** при старте и остановке спама бот присылает отчёт с причиной остановки.

## Доп. команды для развлечения (`/funhelp` — полный список в самом боте)

Работают сразу после деплоя, ничего дополнительно настраивать не нужно.

| Категория | Команды |
|---|---|
| 🎲 Игры | `/dice` `/dart` `/bball` `/bowl` `/slot` `/foot` — анимированные кости Telegram; `/rolldice 2d6+3` — бросок по формуле NdM[+K] |
| 🔮 Развлечения | `/8ball вопрос`, `/coin`, `/joke`, `/fact`, `/rate что-то`, `/ship Имя1 + Имя2`, `/choose вариант1 \| вариант2` |
| ✍️ Текст | `/mock текст`, `/reverse текст`, `/clap текст`, `/vaporwave текст`, `/ascii текст`, `/calc выражение` |
| 🖼 Картинки (ответом на фото/сообщение) | `/meme Текст сверху \| Текст снизу`, `/fry`, `/glitch`, `/quote` (текст → карточка-цитата) |
| 👤 Профиль | `/whois`, `/alive` |
| 📸 Сторис | `/stori <ссылка на сторис>` — скачать конкретный сторис (`https://t.me/user/s/123`); `/allstori @user` (или ответом на сообщение человека в группе) — скачать все доступные сторис из профиля (активные + закреплённые) |
| 💾 Скачивание | `/save <ссылка>` — скачать видео/фото по ссылке из Instagram, TikTok, YouTube, Twitter/X, Pinterest и других соцсетей (через yt-dlp); можно ссылкой-аргументом или ответом на сообщение со ссылкой |
| 👁 Слежка | `/watch @user`, `/unwatch @user`, `/watchlist` — уведомление в Избранное при смене имени/юзернейма/фото |

**Куда летят сторис:** если команда выполнена в группе/канале — скачанное отправляется в Избранное; если в личной переписке с человеком — прямо в эту переписку.

Реализация — в `fun.py`, подключается в `main.py` через `fun.register(...)` и не меняет существующую логику спама/автоответчика/скачивания медиа.

## Автоответчик

**Автоответчик неактивен по умолчанию.** Активируется только после настройки обоих сообщений командами `/setmsg1` и `/setmsg2`.

При первом сообщении от нового человека:
1. Отправляет 1-е сообщение
2. Через 5 секунд (со статусом «печатает...») отправляет 2-е сообщение
3. Ставит на «без звука» и кидает в архив

**Защита от спама в ЛС:**
- 5+ одинаковых сообщений подряд → чёрный список + удаление чата
- 5+ стикеров подряд → чёрный список + удаление чата
